# YouAreDaChef

A Ruby implementation of an YouAreDaChef library inspired by https://github.com/raganwald/YouAreDaChef.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'YouAreDaChef'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install YouAreDaChef
